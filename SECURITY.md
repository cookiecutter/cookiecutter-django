# Security Policy

## Supported Versions

Only the latest version of cookiecutter-django is supported.

## Reporting a Vulnerability

If you think you have found a vulnerability, and even if you are not sure, please [report it to us in private](https://github.com/cookiecutter/cookiecutter-django/security/advisories/new). We will review it and get back to you. Please refrain from public discussions of the issue.
